To install the script follow the steps 1-10

1.          First of all, upload all the files to server, into a new directory

2.         Create a new MySQL database using your website management control panel (eg: Cpanel) and assign a new username and password for it.
   

3.         Open the the config.inc.php and edit some important configurations as described .

4.         You have to enter your license key in the config file.

            Note
            ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

            The license can be generated after logging into the members area of inoutscripts.com.
            Please use your email address and the given password to login.

           ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

5.         Now, set your admin area username and password in the config file. You can also change the admin login username and password anytime in future in config.inc.php.

6.         Now you can enter your MySQL settings in the config.inc.php file as described in it.

 

            The setting for "localhost" usually doesn't need to be changed but your host may have a different name for their MySQL server.

            The settings for "username", "password" and "database name" usually need to be entered like this in the config.inc file. [If you need assistance most hosts will happily guide you.]

                        yourdomain_username

                        password just as it is

                        yourdomain_databasename

 

7.         In config.inc.php change the url for $server_dir= to point to the full path of the directory where you have installed the script,

                        eg. $server_dir="http://mydomain.com/newsletter/"

            [Remember to include the last forward slash.]     



8.         Change the access permission[CHMOD] of the directories attachments,csv,export,import and userfiles to 777

 

9.        Edit the confirm.inc.php file in case you want to send confirmation email when people subscribe to a list.



 10.        Go to the URL http://yourdomain.com/newsletter/index.php in your browser and login using the username and password you set in the config.inc.php file. After logging in

           please  invoke http://yourdomain.com/newsletter/install.php
             

            IMPORTANT NOTE
            ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

            If you are getting any error message in the install page saying some loading file is missing, please find the required file in the zip file
            http://www.inoutscripts.com/downloads/ioncube_loader_files.zip and upload to the 'ioncube' directory.
            If you have any query, please contact our support desk. We will be glad to help you.

            ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

             

            That's it!    
        

     

To upgrade the script follow these steps

           IMPORTANT NOTE
           ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

           If you have active campaigns, there are chances that the campaign may not be fired to some emails in the queue. Also there are chances that the statistics may go wrong.
           It is strongly recommended to upgrade after all active/inactive campaigns are completed/restarted.

           ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1.       Upload the files to your installation directory except config.inc.php. Copy the mysql settings and admin login settings from old config file to the new config file.

          Please update the value of $charset_encoding according to your current system encoding. Until version 4.2 the script was using iso-8859-1. From version 4.3 onwards the deafult encoding will be utf-8.

          Those who are upgrading must specify the encoding correctly to ensure data consistency. If you need any help contact our support desk

          Upload the config.inc.php after making the above changes.

2.       Ensure that the access permission[CHMOD] of the directories csv and userfiles is 777

3.       After logging in please invoke install.php from browser; your tables will be upgraded automatically.
