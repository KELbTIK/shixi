PHPMailer-BMH (Bounce Mail Handler)

PHPMailer-BMH is a PHP program to check your IMAP/POP3 inbox and delete all 'hard' bounced emails. It features a callback function where you can create a custom action. This provides you the ability to write a script to match your database records and either set inactive or delete records with email addresses that match the 'hard' bounce results.
